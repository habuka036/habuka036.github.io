Eucalyptus Advent Calendar 2012 JP 前夜祭

明日から Eucalyptus Advent Calendar 2012 JP が始まります

北瀬さんが CloudStack Advent Calendar jp: 2012 をやると聞いて、「んじゃぁ、うちもやってみよう。だけど CloudStack と違って人気がないから独りで細々とやろう」と気軽に考えて立ち上げた Eucalyptus Advent Calendar 2012 JP ですが、11/30 の時点でなんと 11 人の参加者となっていました。素直に驚きです。

ちなみに、その CloudStack Advent Calendar jp: 2012 は前日の時点で 19 名の参加者で、OpenStack Advent Calendar 2012 JP は 14 人の参加者となっており、さすがに二大巨頭(？)と言えども 25 人は揃わなかったようです。(難しいですね…f^^;)

で、「前夜祭」というタイトルなので何かひとネタと考えていたのですが、実はこれを書いているうちに寝落ちしてたので、今日はこのぐらいで。あ、ちなにみに、誰も参加してくれなかったとき用に 25 日分のネタを考えておいたのですが、折角なので以下にその一覧を。

  1. euca-monkey を使ってみる
  2. eutester を使ってみる
  3. se34euca を使ってみる
  4. image-configs を使ってみる
  5. silvereye を使ってみる
  6. bodega を使ってみる
  7. JEUG版 Live-DVD の作り方
  8. doctor-euca を使ってみる
  9. nagios 連携を試してみる
  10. faststart を使ってみる
  11. 1.6.2, 2.0.3, 3.x 起動時間比較
  12. Walrus だけ使ってみたよ
  13. Walrus の秘孔をついてみる
  14. Web 管理画面のテーマを変えてみる
  15. NC のディスクを GlusterFS で
  16. CC の Image Proxy について
  17. インスタンスは何台まで起動できるか？
  18. 各コンポーネントの通話を盗み聞きしてみよう
  19. 高いストレージを使わずに Walrus と SC を冗長化
  20. Eucalyptus とバックアップツール
  21. AWS と連携してみる
  22. Walrus を Glance に変更してみる？
  23. Splunk でログを集めてみる
  24. ESXi 使ってみる
  25. ゲストOS として ReactOS を動かしてみる
  26. ゲストOS として Plan9 を動かしてみれる？

では、参加してくれた皆さんと、読んでくれる皆さんと、そして参加しようかどうしようか迷っているあなたに感謝しつつ、はりきって 25 日間の祭を楽しみたいと思います。
